# Vidéos HeyGen — Module P01 (Téléchargement Minecraft)

Dépose ici les 7 fichiers MP4 générés via HeyGen (voir `Module1-Jackson-P01-HeyGen-Scripts.md` à la racine du projet pour les scripts).

| Fichier attendu | Scène |
|---|---|
| `hook.mp4` | Accroche (20 s) |
| `concept.mp4` | Concept clé (90 s) |
| `example-1-microsoft.mp4` | Exemple compte Microsoft (30-45 s) |
| `example-2-launcher.mp4` | Exemple launcher (30-45 s) |
| `example-3-premier-monde.mp4` | Exemple premier lancement (30-45 s) |
| `try-it-intro.mp4` | Intro du quiz (15 s) |
| `recap.mp4` | Récap + badge (20 s) |

**Format recommandé** : paysage 16:9, 1080p, MP4 H.264.

**Tant que les vidéos ne sont pas déposées**, le composant AvatarScene affichera un player avec une vidéo 404 — les sous-titres restent visibles, le reste du module (concept cards, exemples textuels, quiz, badge) fonctionne quand même.
